### sub2ind()
convert subscripts to linear indices

 => |(1, 1)|(1, 2)|(1, 3)|
 
    |(2, 1)|(2, 2)|(2, 3)|
    
    |(3, 1)|(3, 2)|(3, 3)|

 => | 1 | | 4 | | 7 |
 
    | 2 | | 5 | | 8 |
    
    | 3 | | 6 | | 9 |

 => | 1 |
 
    | 2 |
    
    | 3 |
    
    | 4 |
    
    | 5 |
    
    | 6 |
    
    | 7 |
    
    | 8 |
    
    | 9 |
